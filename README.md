# Bliss Escola de Dança — Site oficial

Site estático publicado pelo GitHub Pages.

## Galeria

As fotos e vídeos podem ser enviados diretamente para as pastas em `galeria/midia`, sem alterar o código. Consulte `galeria/COMO_ADICIONAR_FOTOS_E_VIDEOS.md`.

## Estrutura principal

- `index.html`: página inicial
- `ritmos/`: páginas individuais dos ritmos
- `galeria/`: galeria de espetáculos
- `assets/`: logos, favicon e elementos visuais

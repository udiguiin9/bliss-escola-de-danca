# Como adicionar fotos e vídeos à Galeria Bliss

1. No GitHub, abra a pasta `galeria/midia`.
2. Entre na pasta do espetáculo desejado.
3. Clique em **Add file → Upload files**.
4. Envie os arquivos respeitando os nomes abaixo e confirme em **Commit changes**.

## Capa

Use `capa.jpg`, `capa.jpeg`, `capa.png` ou `capa.webp`.

## Fotos

Use `foto-01.jpg`, `foto-02.jpg` ... até `foto-30.jpg`. Também são aceitos JPEG, PNG e WebP.

## Vídeos

Use `video-01.mp4` ... até `video-05.mp4`. Também são aceitos WebM e MOV.

O site detecta os arquivos automaticamente; não é necessário editar o HTML.

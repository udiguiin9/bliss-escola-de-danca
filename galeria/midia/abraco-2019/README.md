# ABRAÇO — 2019

Envie as fotos e vídeos deste espetáculo para esta pasta no GitHub.

## Nomes reconhecidos automaticamente pelo site

- Capa: `capa.jpg`, `capa.jpeg`, `capa.png` ou `capa.webp`
- Fotos: `foto (1).JPG` até `foto (100).JPG`
- Vídeos: `video-01.mp4` até `video-05.mp4` (também aceita `.webm` e `.mov`)

## Regra importante para as fotos

Use numeração contínua, sem pular números:

`foto (1).JPG`, `foto (2).JPG`, `foto (3).JPG`...

O site encerra a busca ao encontrar o primeiro número ausente. Portanto, se não existir `foto (8).JPG`, as fotos a partir de `foto (9).JPG` não serão exibidas.

Mantenha todas as fotos com a extensão `.JPG` em letras maiúsculas.
